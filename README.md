## Описание программы

** Random Password Generator - это GUI-приложение которое позволяет быстро создавать надёжные пароли любой сложности с сохранением истории.**

### Возможности:
- ✅ Генерация пароля с выбором длины и из чего будет состоять пароль (число, буква, спецсимвол) 
- ✅ История паролей сохраняется в формате JSON

## Инструкция по запуску

### Требования:
- Python 3.6 или выше
- Установленная библиотека Tkinter 
- Сохранение в JSON

### Запуск:
1. Откройте терминал или командную строку
2. Перейдите в директорию проекта:
3. Открыть файл password.py
4. Запустить приложение python password.py
   

## Примеры использования

###Генерация пароля

1.	Ползунок позволяет выбрать длину пароля (от 4 до 32 символов).
2.	Чекбоксы отвечают за включение/выключение цифр, букв и спецсимволов.
3.	При нажатии кнопки «Сгенерировать пароль» программа проверяет настройки
4.	Сгенерированный пароль отображается в поле ввода.
5.	Пароль автоматически сохраняется в файл password_history.json.
6.	Все сохранённые пароли отображаются в таблице ниже.
